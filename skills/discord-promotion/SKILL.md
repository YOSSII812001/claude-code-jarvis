## 宣伝方法の3カテゴリ

| カテゴリ | 概要 | 手軽さ | リーチ |
|---------|------|--------|--------|
| **サーバー内宣伝** | 宣伝専用Discordサーバーに投稿 | 簡単 | Discord内ユーザー |

## 1. サーバー内宣伝

他のDiscordサーバー内で宣伝する方法。手軽で即効性がある。

> 各サーバーのルールを必ず確認すること。

### Discord ADS
- **招待リンク**: https://discord.gg/6DNAtRMpfa
- **規模**: 1,500人超の大型宣伝サーバー
- **特徴**: 有名サーバーから出来立てサーバーまで幅広い。初心者専用チャンネルあり
- **おすすめ**: 初めてのサーバー宣伝に最適

### 宣伝鯖 / PR Server
- **招待リンク**: https://discord.gg/qKcya2kHee
- **特徴**: 連携制度がある珍しい宣伝サーバー。連携すると専用チャンネルで宣伝可能
- **おすすめ**: 他サーバーとの連携を活かしたい場合

## 3. その他の宣伝方法

### ユーザープロフィールに掲載
- Discordプロフィールにサーバーリンクを設定
- 他サーバーでよく発言する人ほど効果大
- 最も手軽で自然な宣伝方法

### ホームページ（HP）を作成
- サーバー専用のWebサイトを作成
- 管理コストは増えるが、SEO経由の流入が期待できる
- 例: https://yuuuyu.localinfo.jp/

### SNS活用（X/Instagram/YouTube/TikTok等）
- プロフィールにサーバーリンクを掲載
- コンテンツを楽しみつつ宣伝できる一石二鳥
- 個人で楽しみながら活動するのがポイント

### 海外向け掲示板
| サービス | URL |
|---------|-----|
| **discadia** | https://discadia.com/add/ |
| **DS.me (Discords.com)** | https://discords.com/ |

## 注意事項

- **DMでのランダム勧誘は禁止**: 引き抜き行為として処罰対象になる可能性がある
- **各宣伝サーバーのルールを確認**: 投稿頻度・形式・カテゴリ指定など
- **bump/upの定期実行**: 忘れると順位が下がる。リマインダー設定推奨

---

## 推奨導入順序（初心者向け）

```
1. DISBOARD を導入（まずは定番から）
2. ディス速 を導入（日本向けの定番）
3. Discord ADS に宣伝投稿（即効性）
4. Discordプロフィールにリンク設置（無料・簡単）
5. 余裕があれば DCafe / Dicoall / Discope も追加
6. SNS・HPでの告知（長期的な集客基盤）
```
