# 競争分析フレームワーク

## 1.1 Porter's Five Forces（ポーターの5つの力）

### 概要
マイケル・ポーターが開発した業界の競争環境を分析するフレームワーク。業界の収益性と魅力度を評価する。

### 5つの競争要因

```
                    ┌─────────────────┐
                    │ 新規参入の脅威 │
                    └────────┬────────┘
                             │
                             ▼
┌─────────────┐    ┌─────────────────┐    ┌─────────────┐
│ 売り手の   │◄──►│   業界内の競争  │◄──►│ 買い手の   │
│ 交渉力     │    │                 │    │ 交渉力     │
└─────────────┘    └────────┬────────┘    └─────────────┘
                             │
                             ▼
                    ┌─────────────────┐
                    │ 代替品の脅威   │
                    └─────────────────┘
```

### 分析テンプレート

#### 1. 新規参入の脅威（Threat of New Entrants）

| 評価項目 | 低い脅威 | 高い脅威 | 評価 |
|----------|----------|----------|------|
| 参入障壁 | 高い規制、大規模投資必要 | 参入容易、低コスト | |
| 規模の経済 | 大きい | 小さい | |
| ブランド力 | 強いロイヤリティ | 弱い差別化 | |
| 切替コスト | 高い | 低い | |
| 流通チャネル | 確立済み、排他的 | オープン | |

**スコア:** □ 低(1) □ やや低(2) □ 中(3) □ やや高(4) □ 高(5)

#### 2. 売り手の交渉力（Bargaining Power of Suppliers）

| 評価項目 | 低い交渉力 | 高い交渉力 | 評価 |
|----------|------------|------------|------|
| サプライヤー数 | 多い | 少ない（寡占） | |
| 差別化 | 代替容易 | 独自技術・特許 | |
| 切替コスト | 低い | 高い | |
| 前方統合の脅威 | 低い | 高い | |
| 業界の重要性 | 主要顧客 | マイナー顧客 | |

**スコア:** □ 低(1) □ やや低(2) □ 中(3) □ やや高(4) □ 高(5)

#### 3. 買い手の交渉力（Bargaining Power of Buyers）

| 評価項目 | 低い交渉力 | 高い交渉力 | 評価 |
|----------|------------|------------|------|
| 買い手の集中度 | 分散 | 集中（大口顧客） | |
| 購入量 | 小さい | 大きい | |
| 製品の差別化 | 高い | コモディティ化 | |
| 切替コスト | 高い | 低い | |
| 後方統合の脅威 | 低い | 高い | |
| 情報の非対称性 | 情報不足 | 十分な情報 | |

**スコア:** □ 低(1) □ やや低(2) □ 中(3) □ やや高(4) □ 高(5)

#### 4. 代替品の脅威（Threat of Substitutes）

| 評価項目 | 低い脅威 | 高い脅威 | 評価 |
|----------|----------|----------|------|
| 代替品の有無 | 少ない | 多い | |
| 価格性能比 | 劣る | 優れる | |
| 切替コスト | 高い | 低い | |
| 買い手の傾向 | 保守的 | 新しもの好き | |

**スコア:** □ 低(1) □ やや低(2) □ 中(3) □ やや高(4) □ 高(5)

#### 5. 業界内の競争（Industry Rivalry）

| 評価項目 | 低い競争 | 高い競争 | 評価 |
|----------|----------|----------|------|
| 競合の数 | 少ない | 多い | |
| 業界成長率 | 高成長 | 低成長・衰退 | |
| 固定費比率 | 低い | 高い | |
| 製品の差別化 | 高い | 低い | |
| 撤退障壁 | 低い | 高い | |
| 戦略の多様性 | 似ている | 異なる | |

**スコア:** □ 低(1) □ やや低(2) □ 中(3) □ やや高(4) □ 高(5)

### 総合評価

**業界の魅力度 = (25 - 合計スコア) / 20 × 100%**

- 80%以上: 非常に魅力的
- 60-80%: 魅力的
- 40-60%: 普通
- 20-40%: 魅力度低い
- 20%以下: 参入非推奨

## 1.2 SWOT分析 / TOWS分析

### SWOT分析

**内部環境と外部環境を4象限で整理**

```
         │ 好ましい影響    │ 好ましくない影響
─────────┼─────────────────┼──────────────────
内部要因 │ Strengths       │ Weaknesses
         │ 強み            │ 弱み
─────────┼─────────────────┼──────────────────
外部要因 │ Opportunities   │ Threats
         │ 機会            │ 脅威
```

### SWOTの問いかけ

**Strengths（強み）:**
- 他社より優れている点は？
- 独自のリソース・能力は？
- 顧客が選ぶ理由は？
- 成功要因は何か？

**Weaknesses（弱み）:**
- 改善すべき点は？
- 不足しているリソースは？
- 顧客の不満は？
- 競合に劣る点は？

**Opportunities（機会）:**
- 市場のトレンドは？
- 規制変化で生まれる機会は？
- 技術変化の影響は？
- 競合の弱点は？

**Threats（脅威）:**
- 競合の動きは？
- 市場環境の変化は？
- 規制リスクは？
- 技術の陳腐化リスクは？

### TOWS分析（戦略マトリクス）

**SWOTから戦略オプションを導出**

```
                 │ 強み(S)           │ 弱み(W)
─────────────────┼───────────────────┼───────────────────
機会(O)          │ SO戦略            │ WO戦略
                 │ 強みで機会を活かす │ 弱みを克服し機会を活かす
                 │ （積極攻勢）       │ （弱点強化）
─────────────────┼───────────────────┼───────────────────
脅威(T)          │ ST戦略            │ WT戦略
                 │ 強みで脅威を回避   │ 弱みと脅威の最小化
                 │ （差別化）         │ （撤退・縮小）
```

### TOWS戦略の例

| 戦略 | アプローチ | 例 |
|------|------------|-----|
| SO | 強み×機会で攻める | 技術力を活かして成長市場に参入 |
| WO | 弱みを補って機会を取る | パートナーシップで販路不足を解消 |
| ST | 強みで脅威を跳ね返す | ブランド力で価格競争を回避 |
| WT | 損失を最小化 | 不採算事業から撤退 |

### 競合の4つの分類

1. **直接競合** - 同じ顧客に同じ製品を提供
2. **間接競合** - 異なる製品で同じニーズを満たす
3. **潜在競合** - 将来参入してくる可能性がある
4. **代替競合** - 全く異なる方法で問題を解決
