# 財務分析フレームワーク

## 4.1 財務3表分析

### 損益計算書（P/L）の主要指標

| 指標 | 計算式 | 意味 |
|------|--------|------|
| 売上総利益率 | (売上高 - 売上原価) / 売上高 | 製品の収益性 |
| 営業利益率 | 営業利益 / 売上高 | 本業の収益性 |
| 経常利益率 | 経常利益 / 売上高 | 財務活動含む収益性 |
| 当期純利益率 | 当期純利益 / 売上高 | 最終的な収益性 |

### 貸借対照表（B/S）の主要指標

| 指標 | 計算式 | 意味 | 目安 |
|------|--------|------|------|
| 流動比率 | 流動資産 / 流動負債 | 短期支払能力 | 200%以上 |
| 当座比率 | 当座資産 / 流動負債 | 即時支払能力 | 100%以上 |
| 自己資本比率 | 純資産 / 総資産 | 財務の安全性 | 40%以上 |
| D/Eレシオ | 有利子負債 / 純資産 | 負債依存度 | 1倍以下 |

### 収益性指標

| 指標 | 計算式 | 意味 |
|------|--------|------|
| ROE | 当期純利益 / 自己資本 | 株主資本に対するリターン |
| ROA | 当期純利益 / 総資産 | 総資産に対するリターン |
| ROIC | NOPLAT / 投下資本 | 投下資本に対するリターン |

### デュポン分析（ROE分解）

```
ROE = 当期純利益 / 自己資本

    = (当期純利益/売上高) × (売上高/総資産) × (総資産/自己資本)

    = 売上高純利益率 × 総資産回転率 × 財務レバレッジ
      (収益性)        (効率性)      (安全性)
```

## 4.2 DCF法（ディスカウント・キャッシュ・フロー）

### 概要
将来のキャッシュフローを現在価値に割り引いて企業価値を算定

### 計算式

```
企業価値 = Σ (FCFt / (1+r)^t) + TV / (1+r)^n

FCF: フリーキャッシュフロー
r: 割引率（WACC）
TV: ターミナルバリュー（継続価値）
n: 予測期間
```

### フリーキャッシュフロー（FCF）の計算

```
FCF = EBIT × (1 - 税率)
    + 減価償却費
    - 設備投資
    - 運転資本増加
```

### WACC（加重平均資本コスト）

```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tc)

E: 株式価値
D: 負債価値
V: E + D
Re: 株主資本コスト
Rd: 負債コスト
Tc: 法人税率
```

### ターミナルバリュー

**永続成長モデル:**
```
TV = FCFn+1 / (WACC - g)

g: 永続成長率（通常1-3%）
```

## 4.3 投資判断指標

| 指標 | 計算方法 | 判断基準 |
|------|----------|----------|
| NPV（正味現在価値） | 将来CFの現在価値 - 初期投資 | 正なら投資実行 |
| IRR（内部収益率） | NPV = 0となる割引率 | ハードルレート以上なら投資 |
| 回収期間 | 初期投資 / 年間CF | 目標期間内なら投資 |
| PI（収益性指数） | 将来CFの現在価値 / 初期投資 | 1以上なら投資 |
