# 成長戦略フレームワーク

## 2.1 Ansoff Matrix（アンゾフ・マトリクス）

### 概要
製品と市場の2軸で成長戦略を4つに分類

```
              │ 既存製品           │ 新規製品
──────────────┼────────────────────┼────────────────────
既存市場      │ 市場浸透           │ 製品開発
              │ (Market            │ (Product
              │  Penetration)      │  Development)
              │ リスク: 低         │ リスク: 中
──────────────┼────────────────────┼────────────────────
新規市場      │ 市場開拓           │ 多角化
              │ (Market            │ (Diversification)
              │  Development)      │
              │ リスク: 中         │ リスク: 高
```

### 各戦略の詳細

#### 市場浸透（既存製品×既存市場）
**目標:** 既存市場でのシェア拡大

**施策例:**
- 価格戦略（値下げ、バンドル）
- プロモーション強化
- 販売チャネル拡大
- 顧客ロイヤリティ向上
- 競合からの顧客獲得

**適切な状況:** 市場成長中、シェア拡大余地あり

#### 製品開発（新規製品×既存市場）
**目標:** 既存顧客への新製品提供

**施策例:**
- 製品ラインの拡張
- 次世代製品の開発
- 補完製品の追加
- 品質・機能の改善

**適切な状況:** 顧客ニーズが多様、技術力あり

#### 市場開拓（既存製品×新規市場）
**目標:** 新しい市場・セグメントへの進出

**施策例:**
- 地理的拡大（海外進出）
- 新規セグメント開拓
- 新規チャネル活用
- 用途開発

**適切な状況:** 製品力あり、国内市場飽和

#### 多角化（新規製品×新規市場）
**目標:** 新市場に新製品で参入

**タイプ:**
- 関連多角化（シナジーあり）
- 非関連多角化（コングロマリット）

**適切な状況:** 既存事業の成長限界、リスク分散

## 2.2 BCG Growth-Share Matrix（BCGマトリクス）

### 概要
ボストン・コンサルティング・グループが開発した製品ポートフォリオ分析ツール

```
市場成長率
    高│  ★ Star        │  ? Question Mark
      │  花形          │  問題児
      │                │
      │  投資して成長  │  選別投資
      ├────────────────┼────────────────
    低│  Cash Cow      │  Dog
      │  金のなる木    │  負け犬
      │                │
      │  収穫して維持  │  撤退・売却
      └────────────────┴────────────────
              高            低
           相対的市場シェア
```

### 4つの象限

#### Star（花形）
- **特徴:** 高成長市場で高シェア
- **戦略:** 積極投資で市場リーダーの地位を維持
- **キャッシュフロー:** 大きな投資が必要、収益も大きい
- **例:** 成長期のスマートフォン事業

#### Cash Cow（金のなる木）
- **特徴:** 低成長市場で高シェア
- **戦略:** 投資を最小限に、利益を刈り取る
- **キャッシュフロー:** 安定した高収益
- **例:** 成熟市場の主力製品

#### Question Mark（問題児）
- **特徴:** 高成長市場で低シェア
- **戦略:** 選択と集中、投資するか撤退するか判断
- **キャッシュフロー:** 大きな投資必要、収益不安定
- **例:** 参入初期の新規事業

#### Dog（負け犬）
- **特徴:** 低成長市場で低シェア
- **戦略:** 撤退・売却を検討
- **キャッシュフロー:** 最小限
- **例:** 衰退期の旧製品

### BCGマトリクスの限界

- 市場定義が恣意的
- 2軸のみで過度に単純化
- シナジーを考慮しない
- 中間的なポジションを説明できない

## 2.3 Blue Ocean Strategy（ブルーオーシャン戦略）

### 概要
競争のない新市場を創造する戦略（レッドオーシャン vs ブルーオーシャン）

### レッドオーシャン vs ブルーオーシャン

| 項目 | レッドオーシャン | ブルーオーシャン |
|------|------------------|------------------|
| 市場 | 既存市場で競争 | 新市場を創造 |
| 需要 | 既存需要を奪い合う | 新需要を創出 |
| 競争 | 競合を打ち負かす | 競争を無意味に |
| 価値とコスト | トレードオフ | 同時追求 |
| 戦略の焦点 | 差別化 or 低コスト | 差別化 and 低コスト |

### 戦略キャンバス

**横軸:** 業界の競争要因
**縦軸:** 提供レベル（低〜高）

```
提供レベル
    高│       ╱╲      業界平均
      │      ╱  ╲
      │     ╱    ╲
      │    ╱      ╲   ────── 新戦略
      │   ╱        ╲
    低│──╱──────────╲───────────────
      │ 要因A 要因B 要因C 要因D 要因E
```

### ERRCグリッド

| アクション | 説明 | 質問 |
|------------|------|------|
| **E**liminate（排除） | 業界の常識をなくす | 当たり前に提供してきたが、実は不要なものは？ |
| **R**educe（減少） | 業界標準以下に減らす | 過剰提供しているものは？ |
| **R**aise（増加） | 業界標準以上に高める | 顧客が本当に求めているものは？ |
| **C**reate（創造） | 業界になかったものを作る | 全く新しい価値は？ |

### ブルーオーシャンの成功例

| 企業 | 排除/減少 | 増加/創造 |
|------|-----------|-----------|
| Cirque du Soleil | 動物、スター芸人 | 芸術性、テーマ、快適な座席 |
| IKEA | 組み立てサービス、店員 | デザイン、低価格、即持ち帰り |
| Southwest Airlines | 機内食、ラウンジ | 低価格、高頻度運航、楽しさ |
| Nintendo Wii | 高性能グラフィック | 直感操作、家族向け、運動系 |
