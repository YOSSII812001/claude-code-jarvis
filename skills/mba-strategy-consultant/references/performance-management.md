# 実行管理フレームワーク

## 5.1 OKR（Objectives and Key Results）

### 概要
目標（Objective）と主要成果指標（Key Results）で目標管理

### OKRの構造

```
Objective（目標）
├─ Key Result 1（成果指標1）
├─ Key Result 2（成果指標2）
└─ Key Result 3（成果指標3）
```

### 良いOKRの条件

**Objective:**
- 野心的で鼓舞するもの
- 定性的でも可
- 四半期で達成可能

**Key Result:**
- 測定可能（数値で表現）
- 達成困難だが不可能ではない（達成率60-70%が理想）
- 2-5個程度

### OKRの例

**Objective:** 日本市場でNo.1のブランドになる

- **KR1:** NPS（顧客推奨度）を50から70に向上
- **KR2:** 市場シェアを15%から25%に拡大
- **KR3:** ブランド認知度を40%から60%に向上

### OKRカスケード

```
会社OKR
├─ 部門OKR（会社OKRに貢献）
│   ├─ チームOKR（部門OKRに貢献）
│   │   └─ 個人OKR（チームOKRに貢献）
```

## 5.2 Balanced Scorecard（バランスト・スコアカード）

### 概要
財務以外の視点も含めた包括的な業績管理フレームワーク

### 4つの視点

```
┌─────────────────────────────────────────────────┐
│              財務の視点                        │
│   「株主にどう見えるべきか？」                │
│   売上成長、収益性、キャッシュフロー          │
└─────────────────────────────────────────────────┘
                     ↑
┌─────────────────────────────────────────────────┐
│              顧客の視点                        │
│   「顧客にどう見えるべきか？」                │
│   満足度、ロイヤリティ、市場シェア            │
└─────────────────────────────────────────────────┘
                     ↑
┌─────────────────────────────────────────────────┐
│           業務プロセスの視点                   │
│   「何に秀でるべきか？」                      │
│   品質、効率、イノベーション                  │
└─────────────────────────────────────────────────┘
                     ↑
┌─────────────────────────────────────────────────┐
│          学習と成長の視点                      │
│   「変化と改善をどう続けるか？」              │
│   人材育成、組織文化、ナレッジ                │
└─────────────────────────────────────────────────┘
```

### BSCテンプレート

| 視点 | 戦略目標 | 指標（KPI） | ターゲット | 施策 |
|------|----------|-------------|------------|------|
| **財務** | | | | |
| | 売上成長 | 売上高成長率 | +15% | |
| | 収益性向上 | 営業利益率 | 20% | |
| **顧客** | | | | |
| | 顧客満足 | NPS | 70 | |
| | シェア拡大 | 市場シェア | 25% | |
| **業務** | | | | |
| | 品質向上 | 不良率 | 0.5%以下 | |
| | 効率化 | リードタイム | -30% | |
| **学習** | | | | |
| | 人材育成 | 研修時間 | 40h/人 | |
| | エンゲージメント | eNPS | 50 | |

## 5.3 KPIツリー

### 概要
最終目標からKPIを分解し、アクションにつなげる

### KPIツリーの例（売上向上）

```
売上高
├─ 顧客数 × 客単価
│   ├─ 顧客数
│   │   ├─ 新規顧客数
│   │   │   ├─ リード数 × CVR
│   │   │   │   ├─ Webトラフィック × リード獲得率
│   │   │   │   └─ 展示会来場者 × 名刺交換率
│   │   │   └─ 商談数 × 成約率
│   │   └─ 既存顧客継続数
│   │       └─ 前期顧客数 × 継続率
│   └─ 客単価
│       ├─ 購入頻度
│       └─ 1回あたり購入額
```
