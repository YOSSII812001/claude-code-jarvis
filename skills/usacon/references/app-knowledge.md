## ページ別操作ガイド

### 1. ダッシュボード (`/dashboard`)

**主要要素:**
- **企業選択ドロップダウン**: 「対象企業を選択」— 上部にあり、全ページ共通で使用
- **挨拶カード**: 「おはようございます、{企業名}様」+ 日付 + ストリーク日数
- **今日の経営者の問い**: 質問カード + 選択肢ボタン（売上成長率/営業利益率/顧客継続率/キャッシュ残高）+ 自由入力
- **クイックアクション**: 折りたたみセクション（新規企業登録・分析開始へのショートカット）
- **経営の最適解答**: AIが生成した格言 + 「詳細を見る」+ 「再生成する」ボタン
- **最近の活動**: 折りたたみセクション（分析・レポートの履歴）

### 2. 企業情報 (`/companies`)

**主要要素:**
- **検索バー**: 「企業名または業界で検索」
- **フィルター行**: 業界（ドロップダウン）/ 企業規模（ドロップダウン）/ 分析進捗（ドロップダウン）/ 並び替え（ドロップダウン）
- **企業テーブル**: 列 = 企業名 / 業界（バッジ） / 分析進捗（未着手/完了） / 企業規模 / 登録日 / アクション
- **アクションボタン**: 👁 表示 / ✏️ 編集 / 📋 複製 / 🗑️ 削除
- **右上ボタン**: 「更新」「+ 新規登録」

**企業登録フロー:**
- `/companies/new`: ステップ形式（従来型フォーム入力）
- `/companies/new/interactive`: AI対話形式（AIチャットアシスタントが質問しながら企業情報を収集）
- URL自動入力: URLを入力して「Usaconにまかせる」ボタンでAIが企業情報を自動取得

### 3. 分析・評価 (`/processes`)

**主要要素:**
- **企業選択ドロップダウン**: 「対象企業を選択」
- **進捗バー**: 全体進捗（X/4 完了）
- **ステップ表示**: 変革認識 → 成熟度評価 → CSF案 → デジタル戦略（チェックマーク付き）
- **4つの分析カード**:
  - 変革認識（P1-1-1〜P1-1-4の4つのサブ分析を含む）
  - 成熟度評価
  - CSF案作成（重要成功要因の導出 + SWOT分析）
  - デジタル経営戦略（10セクション構成）
- 各カードに「分析を実行」or「結果を確認」ボタン
- **右上**: 「プロセスサイクル」ボタン → `/processes/cycle`

**分析実行の流れ:**
1. 企業を選択
2. 「分析を実行」ボタンをクリック
3. AI分析が開始（数十秒〜数分待機が必要）
4. 完了後「結果を確認」ボタンで結果表示

### 4. 分析レポート (`/reports`)

**主要要素:**
- レポート一覧表示
- `/reports/digital-strategy`: デジタル戦略レポート一覧
- `/reports/digital-strategy/:documentId`: 個別レポート詳細
- 印刷機能: レポート画面から印刷プレビュー
- PDF出力: レポートをPDFとして出力

### 5. 週間レポート (`/weekly-reports`)

**主要要素:**
- **企業選択ドロップダウン**: 「対象企業を選択」
- **「今週のレポートを生成」ボタン**: 右上の青いボタン
- **ナレッジサマリー**: 「サマリーを生成」ボタン（1クレジット消費）
- **レポート履歴**: 日付範囲 / 回答数 / ストリーク / テーマタグ / 「詳細を見る」ボタン

**週次レポートの流れ:**
1. ダッシュボードの「今日の経営者の問い」に毎日回答
2. 1週間分の回答が蓄積される
3. `/weekly-reports` で「今週のレポートを生成」
4. AI がテーマ分類・感情分析・サマリー・提案を生成

### 6. 補助金検索 (`/subsidies`)

**主要要素:**
- **企業選択ドロップダウン**: 企業の業種・地域・事業内容が検索条件に反映
- **UsaCon検索（おすすめ）**: AI分析で最適な補助金を自動提案（3クレジット消費）— 「今すぐ検索」ボタン
- **詳細検索（キーワード・条件指定）**: 折りたたみセクション、キーワード入力欄
- **検索結果**: 補助金カード一覧
- **「お気に入り一覧を見る（組織共有）」ボタン**: → `/subsidies/favorites`

### 7. 設定関連ページ

- **プロフィール** (`/profile`): ユーザー情報の表示・編集
- **設定** (`/settings`): アカウント設定、AI応答設定（トークン数: 5000/20000/30000）
- **テーマ** (`/theme`): テーマカラー変更
- **ダークモード**: テーマページまたは設定からトグル
- **ヘルプ** (`/help`): ヘルプドキュメント
- **About** (`/about`): アプリ情報

### 8. 決済関連ページ

- **料金プラン** (`/pricing`): Standard（¥70,000/月）/ Professional（¥200,000/月）+ 年額プラン
  - 各プランに「申し込む」ボタン → Stripe Checkout に遷移
- **請求管理** (`/billing`): サブスクリプション状態・請求履歴
  - 「Customer Portal」ボタン → Stripe Customer Portal に遷移
- **決済成功** (`/payment/success`): 決済完了後の表示
- **決済キャンセル** (`/payment/cancel`): 決済中止後の表示

### 9. AIアシスタント（コンサル相談）

**チャットは専用ページなし。全ページ共通の右ドロワー（`ChatAssistant.tsx`）として `Layout.tsx` に組み込まれている。**

#### 導線（エントリーポイント）

| 導線 | 場所 | 条件 | 操作 |
|------|------|------|------|
| **ウサコンFAB** | 右下フローティング（全ページ共通） | 常時表示（モバイルは非表示） | クリックでドロワーを開く |
| **UsaconNavigator** | 画面下部に吹き出しで出現 | 全プロセス完了＋ガイダンス作成＋当日問い回答済み | CTAボタン「チャットを開く」 |

- **サイドバーにチャット導線はない**
- FABのTooltip: 「ウサコンに相談」
- FABはドラッグ移動可能。初期位置は右下（bottom: 80, right: 24）
- 未読メッセージ時はパルスアニメーション、通常時は15秒ごとにシェイクアニメーション

#### ドロワー構造

```
右ドロワー（デスクトップ: 1150px / モバイル: 100%）
├── ヘッダー
│   ├── ウサコンアイコン + 「ウサコン」タイトル
│   ├── モード切替: [相談] [サービス] ← ToggleButton
│   ├── 企業選択ドロップダウン（相談モード時のみ必須）
│   └── ×（閉じる）ボタン
│
├── 左パネル: スレッド一覧（360px、PC版のみ）
│   ├── 「新規チャット」ボタン [+]
│   └── スレッド一覧（タイムスタンプ + タイトル + 削除ボタン）
│
└── 右パネル: メッセージエリア（790px）
    ├── メッセージ表示（スクロール可能）
    │   ├── ユーザーメッセージ（右寄せ、青背景）
    │   └── アシスタントメッセージ（左寄せ、灰背景）
    │       └── 感情アイコン付き + Markdown対応
    │
    └── 入力エリア（固定ボトム）
        ├── 📎 ファイル添付ボタン（相談モードのみ）
        ├── テキスト入力欄（Shift+Enterで改行）
        └── 送信ボタン（⬆️）
```

#### 2つのモード

| モード | ヘッダー表示 | 説明 | 企業選択 | ファイル添付 |
|--------|------------|------|---------|-----------|
| **相談（ITC）** | 「相談」 | 企業の変革戦略相談 | **必須** | 可能 |
| **サービス（Product）** | 「サービス」 | 機能説明・Q&A | 不要 | 不可 |

- モード切替時: 入力リセット + メッセージクリア

#### スレッド管理

- **新規チャット**: 左パネル上部「新規チャット」ボタン → メッセージエリアをリセット
- **既存スレッド選択**: スレッド一覧からクリック → 該当スレッドのメッセージ履歴を読み込み
- **スレッド削除**: 各スレッドの削除アイコン → 確認ダイアログ → 削除
- SSEストリーミングでリアルタイム応答

#### アクション実行機能（ActionStatusQueue）

チャット内でAIが「企業URL登録」「分析実行」等の操作を提案・実行する機能:
```
ステータス表示:
┌──────────────────────────────────────┐
│ ✓ 企業情報を検索         (完了)      │
│ ⏳ URLから企業データを抽出 (進行中)   │
│ ⭕ サムネイルをダウンロード (未実行)   │
│                                      │
│ 【確認して実行】 【キャンセル】       │
└──────────────────────────────────────┘
```

#### アシスタントの感情アイコン

応答メッセージに感情アイコンが付く場合がある:
- まもり（保護）/ なみだ（悲しみ）/ はつらつ（動機）
- きらり（希望）/ まとい（集中）/ たしかに（達成）

#### ファイル添付（相談モードのみ）

- 対応形式: PDF, CSV, Excel, 画像, テキスト等
- ドラッグ&ドロップまたはボタンクリックで添付
- 添付プレビュー: ファイル名 + サイズ + 削除ボタン

#### 表示制御
- **クールダウン**: 5分（同じヒント再表示防止）
- **セッション上限**: 10回表示
- **ルート到着後遅延**: 3〜8秒後に出現
- **「今日は表示しない」**: 本日中の表示を抑制
- チャットドロワーが開いている間はNavigator非表示

#### E2Eテスト時のNavigator自動dismiss

> **教訓（autopilot-batch-20260307）**: UsaconNavigatorのポップアップがE2Eテストの操作を妨げる場合がある。テスト開始時にevaluateで自動dismissすること。

```javascript
// Playwright MCP browser_evaluate で実行
() => {
  // UsaconNavigator の吹き出しを非表示にする
  const navigatorElements = document.querySelectorAll('[class*="UsaconNavigator"], [class*="navigator-popup"]');
  navigatorElements.forEach(el => el.style.display = 'none');
  // localStorageでセッション中の再表示を抑制
  localStorage.setItem('usacon_navigator_dismissed', 'true');
}
```

### MUIコンポーネントのセレクタパターン（E2Eテスト用）

> **教訓（autopilot-batch-20260307）**: MUIコンポーネント（ドロワー、Dialog等）の閉じるボタンはsnapshot refが最も確実。data-testidやaria-label指定は不安定な場合がある。

| 操作対象 | 推奨セレクタ | フォールバック | 注意 |
|---------|------------|-------------|------|
| ドロワー閉じる | snapshot refで`×`ボタンを特定 | `[data-testid="CloseIcon"]` → `.closest('button')` | IconButtonの直接クリックは不可、親buttonを取得 |
| Dialog閉じる | snapshot refで`×`ボタンを特定 | `[aria-label="close"]` | MUI Dialogのcloseボタン |
| IconButton全般 | snapshot refが最優先 | `[data-testid="XXXIcon"]` → `.closest('button')` | SVGアイコン直接クリックは無効 |
| ドロワー展開確認 | `[class*="MuiDrawer-paper"]` の存在チェック | `browser_wait_for` でドロワー内テキスト出現待ち | アニメーション完了を待つ |

## 共通操作パターン

### ログイン操作
```
1. https://preview.usacon-ai.com にナビゲート
2. メールアドレス入力欄に ytakeshita@robbits.co.jp を入力
3. パスワード入力欄に password123 を入力
4. 「ログイン」ボタンをクリック
5. /dashboard にリダイレクトされることを確認
```

### 企業選択操作
多くのページで「対象企業を選択」ドロップダウンがある。
```
1. ドロップダウンをクリック
2. 企業名を選択（例: 株式会社ロビッツ3）
3. ページ内容が選択企業に切り替わる
```

### ページ遷移確認
```
1. サイドバーメニューまたはURLで遷移
2. スクリーンショットでページタイトルとコンテンツを確認
3. URLが期待通りであることを確認
```

### チャットドロワーを開く
```
1. 右下のウサコンFABアイコン（丸いキャラクターアイコン）をクリック
2. 右からドロワーが開く（デスクトップ1150px幅）
3. ヘッダーにモード切替「相談」「サービス」が表示される
```

### チャットでメッセージを送信
```
1. ドロワー下部の入力欄にメッセージを入力
2. 送信ボタン（⬆️）をクリック、またはEnterキー
   ※Shift+Enterで改行、IME確定のEnterでは送信されない
3. SSEストリーミングでリアルタイムに応答が表示される
4. 応答完了を確認（ローディング表示が消える）
```

### チャットモード切替
```
1. ドロワーヘッダーの「相談」「サービス」ToggleButtonをクリック
2. 「相談」（ITC）: 企業選択必須、ファイル添付可、戦略相談向け
3. 「サービス」（Product）: 企業選択不要、機能Q&A向け
   ※切替時にメッセージ履歴はリセットされる
```

### チャットでファイル添付（相談モード）
```
1. 「相談」モードに切り替え
2. 入力欄左の📎アイコンをクリック（またはドラッグ&ドロップ）
```

### チャットスレッド管理
```
1. 新規チャット: 左パネル上部の「新規チャット」[+]ボタン
2. 既存スレッド: 左パネルのスレッド一覧からクリック → 履歴読み込み
3. スレッド削除: 各スレッドの🗑️アイコン → 確認ダイアログ → 削除
```

### AI分析・生成の待機
```
1. 「分析を実行」「生成する」等のボタンをクリック
2. ローディング表示を確認
3. wait(time: 30〜120) で十分待機（分析内容により異なる）
4. 結果が表示されることを確認
```
