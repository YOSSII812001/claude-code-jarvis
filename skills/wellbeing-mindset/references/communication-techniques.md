# 実践的コミュニケーション技法

## GROWモデル（詳細）

ジョン・ウィットモア卿らが1980年代後半に開発したエグゼクティブコーチングの基本フレームワーク。

### Goal（目標）
**目的**：会話の方向性と成功基準を明確にする

**質問例**：
- 「この会話から何を得たいですか？」
- 「成功とはどのような状態ですか？」
- 「3ヶ月後、どうなっていたいですか？」
- 「今日話し終わった時、どんな気持ちになっていたいですか？」

**ポイント**：
- SMART目標（Specific, Measurable, Achievable, Relevant, Time-bound）を意識
- 抽象的な目標を具体化する

### Reality（現実）
**目的**：現状を正直に評価し、自己認識を深める

**質問例**：
- 「今、何が起きていますか？」
- 「これまで何を試しましたか？」
- 「何が妨げになっていますか？」
- 「1-10で、目標に対して今どのあたりですか？」
- 「この状況で、あなたが持っているリソースは何ですか？」

**ポイント**：
- 判断せずに聴く
- 事実と解釈を区別する
- 感情も「現実」の一部として扱う

### Options（選択肢）
**目的**：可能性を広げ、創造的に考える

**質問例**：
- 「他にどのような可能性がありますか？」
- 「制約がなければ何を試しますか？」
- 「尊敬する人ならどうするでしょうか？」
- 「これまで試していないことは？」
- 「最悪のアイデアは何ですか？」（逆転発想）

**ポイント**：
- この段階では評価しない
- 「もっとないですか？」と広げる
- 少なくとも3つ以上の選択肢を出す

### Will（意志）
**目的**：具体的な行動約束を引き出す

**質問例**：
- 「何をしますか？」
- 「いつまでにしますか？」
- 「最初の一歩は何ですか？」
- 「1-10で、どれくらいコミットしていますか？」
- 「何がこのコミットメントを妨げる可能性がありますか？」

**ポイント**：
- 具体的で測定可能な行動に落とし込む
- コミットメントレベルが7以下なら、行動を小さくするか障害を探る

## OARSテクニック（モチベーショナルインタビューイング）

### Open questions（開かれた質問）
Yes/Noではなく、詳細を引き出す質問。

**例**：
- ✕「それは辛かったですか？」
- ○「その時、どんな気持ちでしたか？」
- ✕「上司と話しましたか？」
- ○「この状況について、誰かに相談しましたか？」

### Affirmations（肯定）
強みと努力を誠実に認識する。「私は」ではなく「あなたは」を中心に。

**例**：
- ✕「私はあなたを誇りに思います」
- ○「それをやり遂げたのは、本当に強さの表れですね」
- 「これを話題にするのに、勇気が必要でしたね」
- 「困難な中でも前に進もうとしている姿勢が伝わります」

### Reflective listening（反射的傾聴）

**4つの類型**：

| 類型 | 説明 | 例 |
|------|------|-----|
| 単純な反射 | 内容を言い換える | 「締め切りに追われているのですね」 |
| 感情の反射 | 感情を名づける | 「圧倒され、フラストレーションを感じているのですね」 |
| 増幅された反射 | 少し誇張して反論を促す | 「つまり、変化の可能性が全くないということですか」 |
| 両面反射 | アンビバレンスを反映 | 「変わりたい気持ちと、今のままでいたい気持ちと、両方あるのですね」 |

### Summaries（要約）
共有されたことを収集・統合し、オープンな質問で終える。

**構造**：
1. 「ここまでをまとめると...」
2. 主要なポイントを列挙
3. 「何か漏れがありましたか？」

## バリデーション（妥当性確認）の6レベル

マーシャ・ラインハンのDBT（弁証法的行動療法）より。

| レベル | 名称 | 説明 |
|--------|------|------|
| 1 | 存在する | マルチタスクなしで傾聴している |
| 2 | 正確な反射 | 聞いたことを要約して返す |
| 3 | 心を読む | 言葉にされていない感情を表現する |
| 4 | 理解 | 過去の経験の文脈で行動の妥当性を確認 |
| 5 | 正常化 | その感情が当然であることを認識 |
| 6 | 根本的な誠実さ | 脆弱な存在ではなく有能な存在として扱う |

**対話での適用**：
- レベル1-3：基本的な傾聴
- レベル4-5：「その状況なら、そう感じるのは当然です」
- レベル6：「あなたには、これを乗り越える力があると信じています」

## 認知再構成の5ステップ（CBTアプローチ）

### Step 1: 状況の特定
トリガーとなった出来事を具体的に記述する。
- いつ、どこで、誰と、何が起きたか

### Step 2: 自動思考の特定
心に浮かんだことは何か、信念強度を0-100%で評価。
- 「その時、頭に浮かんだ考えは？」
- 「その考えをどれくらい信じていますか？」

### Step 3: 感情と行動の特定
感情とその強度、結果としての行動を記録。
- 「その時、どんな感情がありましたか？」
- 「その結果、何をしましたか？」

### Step 4: 証拠の評価
思考を支持する証拠と反証する証拠を検討。
- 「この考えを支持する証拠は？」
- 「この考えに反する証拠は？」
- 「他の説明はありますか？」

### Step 5: バランスの取れた思考
より現実的でバランスの取れた視点を構築。
- 「よりバランスの取れた考え方は？」
- 「親しい友人が同じ状況なら、何と言いますか？」

## ソクラテス式質問法

### 6カテゴリー

| カテゴリー | 目的 | 例 |
|-----------|------|-----|
| 明確化 | 理解を深める | 「それはどういう意味ですか？」 |
| 仮定を探る | 前提を検証 | 「ここで何を前提としていますか？」 |
| 証拠を探る | 根拠を確認 | 「この信念を支持する証拠は？」 |
| 視点 | 別の見方を促す | 「他の人はこの状況をどう見るでしょうか？」 |
| 含意を探る | 結果を考える | 「それが本当なら、何を意味しますか？」 |
| 質問についての質問 | メタ認知 | 「この考えが繰り返し浮かぶのはなぜだと思いますか？」 |

## 経営者に多い認知の歪みと対応

| 歪み | 典型的な思考 | 対応する質問 |
|------|-------------|-------------|
| 完璧主義 | 「完璧でなければ失敗」 | 「十分に良いとは何%ですか？」 |
| インポスター症候群 | 「このレベルにふさわしくない」 | 「ここに所属するには何を達成する必要が？」 |
| 破局化 | 「この四半期を逃せば終わり」 | 「前回目標を逃したとき何が起きましたか？」 |
| 「すべき思考」 | 「助けなしで対処すべき」 | 「そのルールはどこから来ましたか？」 |
| 心の読み過ぎ | 「皆が私を批判している」 | 「その証拠は何ですか？」 |
| 過度の一般化 | 「いつも失敗する」 | 「成功した時のことを教えてください」 |

## 危機対応の対話例

### 検出した場合のステップ

**Step 1: 認識**
```
「あなたが今、とても辛い状況にいることが伝わってきます。
話してくださってありがとうございます。」
```

**Step 2: 深刻さの確認**
```
「今の気持ちをもう少し聞かせてください。
自分を傷つけたいという気持ちはありますか？」
```

**Step 3: 安全の確保**
```
「あなたの安全が最も重要です。
今、安全な場所にいますか？」
```

**Step 4: 専門家への橋渡し**
```
「私はAIであり、この状況には人間のサポートが必要です。
信頼できる誰かに連絡するか、専門家にご相談ください。
[危機対応リソース]に連絡することをお勧めします。」
```
